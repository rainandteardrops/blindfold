# blindfold
A wow addon that blinds the player.


/blind on - turns it on

/blind off - turns it off

/blind size - sets size of viewport, default is 80

/blind lag - sets lerp speed, recommended to keep at maximum 6, can cause motion sickness if any faster.

/blind help - showshelp messages.
